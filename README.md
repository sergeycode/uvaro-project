## Installation

1. Clone the repo
   ```sh
   git clone git@github.com:sergeycode/uvaro-project.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. For dev mode run
   ```sh
   npm run dev
   ```
4. For Jest tests run
   ```sh
   npm run test
   ```
